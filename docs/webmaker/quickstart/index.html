1: Before you start

1.1: Downloading

There are two offical ways to install Website Maker. There are also multiple versions of Website Maker. This tutorial focuses on the Windows command-line version, but most advice does carry over to the universal command-line version.

With that sorted, let's go downloading! First go to either the Github page and download WebsiteMaker.exe or webmaker.py

1.2: Setup

Now make a file, called template.html. Type out your header code as usual, but when you reach title, don't type a title. Just type {} instead. Continue typing in stuff like linking CSS or favicons, copyright info and anything else every site needs. When you reach the place the title of the page (not the website title), for example, the "Quickstart" text at the top of this page, start the opening tag, in this case h2, then type {} again. Close the tag, then open the p tag. Type {} again, then close the p tag. Add in the footer, then finish your HTML tags.

Now, copy and 1 the file you downloaded into the folder with the files you created earlier.

2: Usage

2.1: Windows

2.1.1: Reccommended

Now, if you're are using the .exe version, double-click the file.

Follow the instuctions onscreen and open the resulting file.

2.1.2: Raw Python

You can use the raw Python files (downloadable from the same link as before), but you need Python 3 installed on your PC. Just move the downloaded file to the same folder as the headertemp files, then double-click the file.

2.2: macOS and Linux

2.2.1: Reccommended

Make sure you've downloaded the webmaker.py file and have copied it to the directory with the headertemp files. Open a terminal window in the folder with the files, or, if you use macOS, follow these steps: 1

  1. Open Spotlight with Command+Space bar and search for Terminal.
  2. Use the cd command to go to the directory with the Python file. In the photo it's the Documents folder.
  3. Type python3 WebsiteMaker.py into the terminal. If you see a message like the one in the screenshot, press Install and wait for it to install.
  4. When it's done, type python3 WebsiteMaker.py again. If it crashes, check that the headertemp files are in the same folder as the downloaded files. Check that a message asking for the filename of the file you want to create appears. If it does you've used it correctly!


^ 1 On Windows and most Linux distros, use Ctrl+C. On Mac systems, use Command+C. To paste, do the same but use V instead of C.

^ 2 You can do the same steps in Linux too, but the VM I had wasn't working as well as I wanted
